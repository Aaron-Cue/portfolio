Proyectos

ALinks

Alinks es una aplicación web desarrollada con React y Firebase que permite a los usuarios registrarse, gestionar sus propios enlaces y compartir un perfil público con sus links destacados. Utiliza Firebase Authentication para el login con autorregistro, Firestore para el almacenamiento de datos, React Router DOM para la navegación, protegida basada en el estado de autenticación.

Weather App

aplicación web desarrollada con React que permite buscar una ciudad y consultar su clima actual, el pronóstico diario para la semana y el pronóstico horario del día. Utiliza una API para obtener las coordenadas a partir del nombre de la ciudad y otra para obtener los datos del clima. Está optimizada con useCallback y useMemo para evitar renders innecesarios, y cuenta con un hook personalizado para manejar las peticiones y respuestas de las APIs.

Cli Task

Aplicación CLI para gestionar tareas, desarrollada en Node.js y conectada a una base de datos MongoDB. Permite realizar operaciones CRUD (crear, leer, actualizar y eliminar) desde la terminal. Utiliza commander para el manejo de comandos, inquirer para entradas interactivas del usuario, picocolors y chalk para mejorar la visualización con estilos y colores en la consola.

Todo App

Aplicación web para gestionar tareas, con una interfaz rápida, simple e intuitiva. Permite agregar tareas con título, descripción, fecha límite y prioridad; marcarlas como completadas, editarlas, eliminarlas y filtrarlas por prioridad. Los datos se guardan automáticamente en localStorage para conservar las tareas entre sesiones. Utiliza un custom hook que combina useReducer con localStorage para manejar el estado, useContext para acceso global, useEffect para sincronización y useRef para referencias en formularios.
